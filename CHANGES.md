## 2.1.18

* mode1(연속동작)에서 간혹 무한루프에 빠져, 안전장치 추가 + 동작 종료 반응속도를 높임
* 반응 속도를 올려서 메모리를 조금 더 사용할 수 있음(약 1MB)
* 매크로 동작 후 숨김 아이콘에서 종료를 클릭해도 프로그램이 꺼지지 않는 문제가 있어서 매크로 강제 종료와 키보드 훅을 해제하는 코드를 강화
* 일부 하드코딩 부분 변경

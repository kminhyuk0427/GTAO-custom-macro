# GTA 매크로


##  주요 기능

-  **mod설정**: 0= 매크로 1회 진행. 1= 누르는 동안 반복 동작됨

##  주의사항

-  **중복 X**: 매크로 진행중에는 다른 매크로가 눌리지 않음
-  **반복 중단**: 매크로 반복동작 중에 키를 때면 그 즉시 동작중이던 매크로가 중지됨
-  **씹덕 주의**: 아이콘 귀여움

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 프로그램 실행
```bash
python main.py
```

### 3. 실행 파일 생성
```bash
build.bat
```

생성된 `dist/GameMacro.exe`를 바탕화면으로 복사
